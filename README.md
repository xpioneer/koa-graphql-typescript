#koa-graphql-typescript

### 项目主要技术结构

***
* koa2
* graphql
* typescript
* ioredis
* mysql2
* mongodb
* koa-router

###安装
***
在终端下操作

项目地址: (`git clone`)

```
git clone git@github.com:xpioneer/koa-graphql-typescript.git
```

安装node_modules依赖 `yarn`

```
yarn #in your command terminal
```
***


###运行
启动开发服务: (http://localhost:8020)

```
yarn start
```

生产环境打包

```
yarn build
```
